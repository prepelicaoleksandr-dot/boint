# Boint – Product Page

Frontend проект страницы товара интернет-магазина.

## 🔗 Live Demo
https://boint-eight.vercel.app/

## 📌 Описание

Проект представляет собой адаптивную страницу товара интернет-магазина.  
Основная цель проекта — практика современной HTML/CSS верстки и создание чистого пользовательского интерфейса.

## 🚀 Функциональность

- адаптивная верстка
- современный UI
- карточка товара
- галерея изображений
- чистая структура HTML
- оптимизированные стили

## 🛠 Стек технологий

- HTML5  
- CSS3  
- JavaScript
